# Student Course Management System

This is a **Student Course Management System** developed as part of an Object-Oriented Programming (OOP) project. The system is designed to help manage student and course-related activities. It allows administrators and users to interact with the system through a web-based interface, where CRUD (Create, Read, Update, Delete) operations are used for managing student data, course enrollment, and more.

## Project Overview

The project aims to implement a fully-functional student course management system using Object-Oriented Programming concepts such as **encapsulation**, **inheritance**, **polymorphism**, and **abstraction**. The backend is implemented using Java and JSP/Servlets, while the frontend uses HTML, CSS (with Bootstrap or Tailwind CSS), and JavaScript.

### Features:

- **Student Management:** Add, update, delete, and search for students.
- **Course Management:** Manage courses, including course registration and course details.
- **Enrollment Management:** Students can enroll in or withdraw from courses.
- **Admin Dashboard:** Admins can view, update, and delete records of students and courses.

## Technologies Used

- **Java**
- **JSP & Servlets**
